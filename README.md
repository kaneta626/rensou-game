# rensou-game
word2vecで連想ゲーム作ってみた  


概要：  
word2vecを用いて、2つの単語がどうやって連想されていくのか調べてみた  
連想をword2vecで実装し、連想結果をwordnetのhypernym()で調べる  


使い方:  
1.app.pyを実行
2.http://127.0.0.1:5000/を開く
3.二つの単語を入力して、実行ボタンを押す
